# To-Do List
 ## :rocket: Tecnologias
Esse projeto foi desenvolvido com as seguintes tecnologias:
* HTML5
* CSS3
* JavaScript

## :computer: Projeto
Este projeto consiste em uma lista de tarefas com as funcionalidades básicas como:
* Adicionar tarefa
* Excluir tarefa
* Marcar como concluído
